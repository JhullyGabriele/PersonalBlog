# Personal Blog

A **personal blog** created with **HTML and CSS**.  

Originally developed in **2016**, when I was under 14 years old. In **2026**, the site was updated to improve **responsiveness** for modern screens.

## How to Run
1. Open the project folder.  
2. Select the file **`inicio.html`**.  
3. Open it in your **browser** to view the blog.

## Notes
- Fully static site using HTML and CSS.  
- Responsive design improvements made in 2026.
